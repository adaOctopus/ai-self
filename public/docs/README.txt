Bundle PDFs (offline Session 1 excerpts)
========================================

Copy your three PDFs into this folder using exactly these filenames:

  the-code-of-cadmus-ebook.pdf
  tldr-dark-mode-busy-professionals.pdf
  self-development-inner-work-workbook.pdf

The app reads them in the browser only (no upload) when the OpenAI API is
unavailable, to add short matched excerpts to offline acknowledgments and
the fallback Cadmus Plan.

Source names on disk may differ; rename on copy to match the list above.
Filenames are listed in lib/pdfBundleManifest.ts if you need to change them.
